5-Layer Fractal Architecture

Self-similar structure at all depths. Every module has the same 5 layers.

▣ Bootstrap

Composition root & DI. Wires everything together at startup. Entry point of your application.

▤ Operators

Business orchestration. Calls Ports, never Implementations. Pure business logic layer.

▥ Core Abstractions

Entities, Ports (interfaces), Events. The heart of your domain. Technology-independent.

▦ Implementations

Technology adapters. Database, API clients, external services. Can nest infinitely.

▧ Boundary

External contracts. DTOs for API requests/responses. Single source of truth for external world.

◈ Fractal Pattern

Same 5 layers at any depth. Only Implementations can contain sub-modules. Infinite nesting enabled.

Operators → Boundary + Core Abstractions
Implementations → Core Abstractions (+ Boundary for system edge adapters)
No circular dependencies. Clean separation. AI-friendly structure.
